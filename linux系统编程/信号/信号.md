# 信号

## 信号的状态

信号有三种状态：产生、未决和递达。

## 信号的处理动作

- 执行默认动作
- 忽略信号
- 捕捉信号。

## 信号相关函数

### signal

函数作用：注册信号捕捉函数。

函数原型：
typedef void(*sighandler_t)(int)
sighandler_t signal(int signalum. Sighandler_t handle );

## kill函数

给指定进程发送信号

kill函数原型：int kill(pid_t pid, int sig );

函数返回值：

- 成功返回0；
- 失败返回1；

函数参数：

- sig信号参数
- pid> 0 发送信号给指定进程。
- pid=0 发送信号给调用函数进程属于同意进程组的所有进程。
- pid<-1 取|pid|发送给对应进程组。
- pid=-1； 发送给进程有权限发送的系统中的所有进程。
